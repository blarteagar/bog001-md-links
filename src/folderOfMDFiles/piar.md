# MDLinks Library
Patricia Arteaga
Venezuela

[Markdown](https://es.wikipedia.org/wiki/Markdown) es un lenguaje de marcado ligero muy popular entre developers. 
Crear una herramienta usando [Node.js](https://nodejs.org/), que lea y analice archivos en formato `Markdown`, para verificar los links que contengan y reportar algunas estadísticas.

[Node.js](https://nodejs.org/es/) es un entorno de ejecución para JavaScript,
construido con el motor de JavaScript [V8 de Chrome](https://developers.google.com/v8/).
